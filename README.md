# Generic Parser - UI

[Try it here](https://generic-parser.netlify.com)

## Getting started

- Install the dependencies
  ```
  npm install
  ```
- Run the application
  ```
  npm dev
  ```
- Build the application
  ```
  npm build
  ```
- Run the production build locally
  ```
  npm preview
  ```
